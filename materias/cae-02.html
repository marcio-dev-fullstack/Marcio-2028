Canais Oficiais do CAE (FNDE)

Os sites oficiais do CAE são gerenciados pelo FNDE e variam conforme a necessidade de consulta, cadastro ou estudo.

  • 1. Portal Principal (Informações e Legislação)
    Para entender as regras, baixar cartilhas e ver notícias atualizadas:
    Página do CAE no Portal Gov.br/FNDE
  • 2. CAE Web (Gestão e Cadastro)
    Sistema para verificar a situação do conselho, vigência do mandato e cadastro de membros.
    Acesso ao CAE Web / SIGPNAE
    Nota: Para consultar o "espelho" do conselho local, use a opção de Consulta Pública no portal.
  • 3. Prestação de Contas (SiPC)
    Para análise de contas e emissão de pareceres:
    Sistema de Prestação de Contas (SiPC) - Contas Online
  • 4. Consulta de Repasses
    Para monitorar os valores transferidos à prefeitura para a merenda:
    Consulta de Liberações de Recursos
💡 Dica Técnica: Como você trabalha com sistemas municipais e bancos de dados como o PostgreSQL, lembre-se que o FNDE costuma disponibilizar dados abertos. Para integrar esses dados em sistemas de fiscalização ou painéis de BI, utilize o Portal de Dados Abertos do FNDE.
